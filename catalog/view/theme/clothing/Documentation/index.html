A)(i) Uploading and installing the theme - top

1. Open the folder named "Theme" and upload the folder "clothing" to your Opencart Installation in catalog/view/theme.

2. Login to the admin section and go to System/Settings and click "edit". See picture below..

3. Select the theme from the "store" tab. Select "apparel".

4. Set the different image sizes. Go to the "image" tab.
