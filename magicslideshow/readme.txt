INSTALLATION:

NOTE: To manually install Magic Slideshow please read 'readme_manual.txt'.

1. Unzip the contents of the zip file and upload them to your OpenCart directory.

2. Go to http://your.site.url/magicslideshow/ in your browser and follow the instructions.

3. Open the 'Extensions' page in your OpenCart admin to activate and customize module.

4. You've now installed the demo version of Magic Slideshow!

5. To upgrade, buy Magic Slideshow and overwrite the magicslideshow.js file with the same file from the full version.
